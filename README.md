# Практическая работа №1 — контейнеризация

В этом задании я упаковала веб-приложение (Go API + Nginx + PostgreSQL) в контейнеры. Использовала многоэтапный Dockerfile для сборки образов и Docker Compose для оркестрации. Также настроил CI/CD через GitHub Actions — теперь образы автоматически улетают в GitHub Container Registry при пуше.

Подробный отчёт со скриншотами и выводами команд Docker приведён в [REPORT.md](REPORT.md).

## Итоги

Локально приложение спокойно поднимается через `docker compose`. Базовая инициализация PostgreSQL через скрипты работает. Сборка и публикация образов в GHCR автоматизирована.
